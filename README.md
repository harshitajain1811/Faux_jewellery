# Artificial Jewellery E-Commerce Website
The platform showcases a curated collection of fashion jewellery developed using mordern tech stacks like React + Vite, Tailwind CSS, Supabase, Integrated with Stripe API.

# Features  
* Responsive design for desktop, tablet, and mobile devices
* Interactive user interface with smooth navigation
* Modern and visually appealing layout
* Optimized for user experience and accessibility

# Purpose
This project was developed as part of my web development portfolio to demonstrate front-end development skills, responsive design principles, and the ability to create visually engaging e-commerce interfaces.
